# fanq

## 核心功能：
极简的倒计时时钟，支持平滑的圆形进度条动画。
白噪音控制板：用户可以通过滑块独立调节雨声、篝火、风铃、咖啡馆背景音的音量，混合出自己最喜欢的专注氛围。
每日专注时长自动保存至 LocalStorage，并以简约的 SVG 趋势图表展示。

## 技术栈：
HTML/CSS/JS 或 React + TailwindCSS + Web Audio API (或使用 howler.js 处理音频)。

## 氛围亮点：
全屏背景采用动态流体渐变（Auroa Background）。
毛玻璃面板（backdrop-filter: blur()），卡片边缘带有极细的渐变发光描边。
